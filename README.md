## 🏸 테니스 예약 시스템

> **서울/경기 테니스장 통합 예약 플랫폼**  
> 사립/공립 구분 없이 빈 코트를 쉽게 찾고 예약할 수 있는 웹 서비스  
> 실시간 알림, 고도화된 검색 필터, 커뮤니티 기능까지 확장 예정

---

## 🎯 최종 목표

- 종합 테니스 플랫폼 구축  
  → **예약 + 중고거래 + 커뮤니티 기능** 통합 제공
- **서울시 내 빈 시간대 코트를 한 눈에 확인하여 예약 가능하게** 만들고 빈자리 알림을 받을 수 있게 하는 것이 핵심

---

## 🧩 핵심 기능 구성

- 🗓️ 테니스장 등록/예약 시스템 (사립 및 공공)
- 📬 실시간 알림 기능 (예약 오픈, 빈자리 발생)
- 🔍 고도화된 검색 필터 (위치, 시간대, 코트 종류 등)
- 🛍️ 중고거래 기능 (예정)
- 🗣️ 커뮤니티 기능 (예정)

---

## 🚀 개발 스프린트 계획

| 단계 | 목표 |
|------|------|
| **1차 스프린트** | 외부 관리자가 **사립 테니스장 등록을 요청**하고, 사용자가 예약할 수 있는 **핵심 예약 시스템** 구축 |
| **2차 스프린트** | 예약 기능 고도화: **알림 기능** , **검색 필터 정교화** , **빈 시간대만 보여주는 시스템** |
| **3차 스프린트** | **서울시 공공서비스 API**를 활용한 빈자리 예약 알림 시스템 구축 → **실서비스 배포 및 운영 시작** |
| **4차 스프린트** | 서울시 공공 API에서 제공되지 않는 테니스장 정보는 **크롤링 기반으로 보완** |
| **5차 스프린트** | 서비스 운영 안정화 후, **커뮤니티 기능 및 중고거래 기능 추가** |

---

## 🧩 문제와 해결 방향

### ❗ 문제점

- 서울시 공립 코트는 여러 사이트에 흩어져 있어 예약이 불편
- 빈자리 확인을 위해 매번 로그인 후 수동 검색 필요
- 취소된 예약 알림이 없음 
- 예약 동시성/트래픽 처리 어려움

### ✅ 해결 전략

- 통합 코트 정보 수집 및 제공 (공공 API + 크롤링)
- 원하는 코트/시간대에 대한 **예약 알림 시스템**
- 실시간 필터 기반 **빈자리 조회 UI**
